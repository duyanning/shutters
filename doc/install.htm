Checkout & Build

(tortoiseSVN and Visual C++ 2005 needed) 

Using TortoiseSVN, fill as below (assuming you checkout it to D:\Shutters)

Add following path to Tools | Options... | Projects | VC++ Directories | Include files and Library files respectively:

D:\Shutters

D:\Shutters\lib

Open examples.sln under folder examples, choose Build | Build Solution.
